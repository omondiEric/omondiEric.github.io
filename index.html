Projects

Firestone

Semester-long group project aimed at implementing an emulation of the popular game Hearthstone, built in Clojure. Clojure a functoinal programming language -> focussed pure functoins to aid in reductoin of unintended complexity. - Agile development to mimic real-world software development life cycle scenario ->bi-weekly sprints, good code design, appropriate abstractions,. - Main topics/skills -> persistent data structures and data consistency, designing for testability, state management, concurrency.

FAT File System

Developed a FAT-like file system to support the creation, deletion and modification of files and directories, keeping the general structure similar to the Microsoft FAT design. project 

Unix Utilities

Implemented "lite" versoins of some important Unix utility functions in relation to the storage and access of data. Gained familiarity with: - linux man pages, Unix file systems, debugging with GDB project 

Where You Art

Semester-long group project aimed at designing an application that will aid museum visitors to explore past installations in real time. It covered all phases of the user experience design lifecycle, including refining  design concepts by gathering user data and iterating on interactive prototypes. I worked with two other team members to design a mobile application for museums, that would enable visitors to access more information about various installations based on which part of the museum they are in. Users would also be able to obtain information on past installations that are not currently on display.
